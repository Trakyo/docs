## Base URL

```
https://api.trakyo.io/v1
```

## Authentication

All endpoints require a Bearer token in the `Authorization` header:

```
Authorization: Bearer tky_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

See [Authentication](/authentication) for details on obtaining and managing API keys.

## Response format

Successful responses return JSON with endpoint-specific fields.

Error responses use [RFC 9457 Problem Details](/concepts/errors) with the `application/problem+json` content type.

## Link identifier

Links are identified by a composite key of `domainId` (UUID) + `key` (slug). Use the sentinel domain ID `00000000-0000-0000-0000-000000000000` for links on the default `trakyo.link` domain.
