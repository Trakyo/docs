---

The Trakyo API uses [RFC 9457 (Problem Details for HTTP APIs)](https://www.rfc-editor.org/rfc/rfc9457.html) for all error responses. This is the same standard used by Cloudflare, Stripe, and other major APIs.

## Error format

Every error response uses the `application/problem+json` content type and includes these fields:

```json
{
  "type": "https://trakyo.io/docs/errors/not_found",
  "title": "Not found",
  "status": 404,
  "detail": "No link found with key 'abc123' on domain '00000000-0000-0000-0000-000000000000' for this account.",
  "instance": "/api/v1/links/00000000-0000-0000-0000-000000000000/abc123/clicks"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `type` | string | URI identifying the error. Links to this documentation. |
| `title` | string | Short human-readable summary. |
| `status` | integer | HTTP status code. |
| `detail` | string | Detailed explanation of what went wrong. |
| `instance` | string | The request path that caused the error (when applicable). |

## Error codes

### 400 -- Bad Request

Invalid parameters or request body.

```json
{
  "type": "https://trakyo.io/docs/errors/bad_request",
  "title": "Bad request",
  "status": 400,
  "detail": "Invalid period 'weekly'. Must be 'all' or '24h'."
}
```

### 401 -- Unauthorized

Missing or invalid API key.

```json
{
  "type": "https://trakyo.io/docs/errors/unauthorized",
  "title": "Unauthorized",
  "status": 401,
  "detail": "Invalid API key."
}
```

### 401 -- Key Expired

API key has expired.

```json
{
  "type": "https://trakyo.io/docs/errors/key_expired",
  "title": "API key expired",
  "status": 401,
  "detail": "API key has expired. Generate a new key in your Trakyo dashboard."
}
```

### 403 -- Forbidden

API key doesn't have access to this resource.

```json
{
  "type": "https://trakyo.io/docs/errors/forbidden",
  "title": "Forbidden",
  "status": 403,
  "detail": "Account not found for this API key."
}
```

### 404 -- Not Found

Resource doesn't exist or doesn't belong to your account.

```json
{
  "type": "https://trakyo.io/docs/errors/not_found",
  "title": "Not found",
  "status": 404,
  "detail": "No link found with key 'abc123' on domain '00000000-0000-0000-0000-000000000000' for this account.",
  "instance": "/api/v1/links/00000000-0000-0000-0000-000000000000/abc123/clicks"
}
```

### 429 -- Rate Limited

Too many requests. See [Rate Limiting](/concepts/rate-limiting).

```json
{
  "type": "https://trakyo.io/docs/errors/rate_limited",
  "title": "Rate limited",
  "status": 429,
  "detail": "Rate limit exceeded. Max 120 requests per minute."
}
```

### 500 -- Internal Error

Something went wrong on our end. If this persists, contact support@trakyo.io.

```json
{
  "type": "https://trakyo.io/docs/errors/internal_error",
  "title": "Internal error",
  "status": 500,
  "detail": "An unexpected error occurred."
}
```
