## 400 — Bad Request

Your request could not be processed because it contains invalid parameters or a malformed body.

### Common causes

- Missing a required field (e.g., `url` when creating a link)
- Invalid field value (e.g., non-UUID `domainId`, unsupported `period`)
- Mixing mutually exclusive parameters (e.g., `period` with `startDate`/`endDate`)
- Empty request body on `POST` or `PATCH` endpoints

### Example response

```json
{
  "error": {
    "code": "bad_request",
    "message": "Missing required field: url",
    "doc_url": "https://trakyo.io/docs/errors/bad_request",
    "request_id": "req_3f02586b7266c812ce7f7937bb9c9e8e",
    "param": "url"
  }
}
```

The `param` field names the offending field when one applies.

### How to fix

Check the `message` field for specifics on what was wrong. If `param` is present, it identifies the exact field that failed validation. Review the [API Reference](/api-reference/introduction) for required parameters and accepted values.
