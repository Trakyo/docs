## 500 — Internal Server Error

An unexpected error occurred while processing your request. This is our fault, not yours.

### Example response

```json
{
  "error": {
    "code": "internal_server_error",
    "message": "An unexpected error occurred.",
    "doc_url": "https://trakyo.io/docs/errors/internal_server_error",
    "request_id": "req_6b14..."
  }
}
```

### How to fix

Retry the request. If the error persists, contact support@trakyo.io with:

- The endpoint you called
- The approximate time of the request
- The `request_id` from the error response
- The full error response body
