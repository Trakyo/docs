## 401 — Key Expired

Your API key was valid but has passed its expiration date.

### Example response

```json
{
  "type": "https://trakyo.io/docs/errors/key_expired",
  "title": "API key expired",
  "status": 401,
  "detail": "API key has expired. Generate a new key in your Trakyo dashboard."
}
```

### How to fix

Generate a new API key from your [Trakyo dashboard](https://app.trakyo.io). The old key cannot be reactivated.
