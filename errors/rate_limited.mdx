## 429 — Rate Limited

You have exceeded the maximum number of requests allowed per minute.

### Limits

| Tier | Limit |
|------|-------|
| Default | 120 requests per minute per API key |

### Example response

```json
{
  "type": "https://trakyo.io/docs/errors/rate_limited",
  "title": "Rate limited",
  "status": 429,
  "detail": "Rate limit exceeded. Max 120 requests per minute."
}
```

### How to fix

Wait before retrying. Implement exponential backoff in your client. If you consistently need higher limits, contact support@trakyo.io.

See [Rate Limiting](/concepts/rate-limiting) for best practices.
