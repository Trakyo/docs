## Base URL

All API requests use the following base URL:

```
https://api.trakyo.io/v1
```

## Endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /links` | List all short links with pagination and filtering |
| `POST /links` | Create a new short link |
| `GET /links/{domainId}/{key}` | Get a single short link |
| `PATCH /links/{domainId}/{key}` | Update a short link |
| `DELETE /links/{domainId}/{key}` | Delete a short link |
| `GET /links/{domainId}/{key}/clicks` | Get click count (all-time, preset period, or custom date range) |
| `GET /domains` | List all shortlink domains |

If you need access to data not yet available via the API, reach out to support@trakyo.io.
