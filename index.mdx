## Base URL

All API requests use the following base URL:

```
https://app.trakyo.io/api/v1
```

## Current Endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /links/{domainId}/{key}/clicks` | Get click count for a short link (all-time or last 24h) |

More endpoints are coming soon. If you need access to data not yet available via the API, reach out to support@trakyo.io.
