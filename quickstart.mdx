## 1. Get your API key

API keys are available from your Trakyo dashboard, or by contacting support@trakyo.io.

Your key looks like this:

```
tky_live_1c126b6be11622008e729749f963c425
```

## 2. Make your first request

<CodeGroup>
```bash curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://app.trakyo.io/api/v1/links/YOUR_DOMAIN_ID/YOUR_LINK_KEY/clicks"
```

```python Python
import requests

response = requests.get(
    "https://app.trakyo.io/api/v1/links/YOUR_DOMAIN_ID/YOUR_LINK_KEY/clicks",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://app.trakyo.io/api/v1/links/YOUR_DOMAIN_ID/YOUR_LINK_KEY/clicks",
  { headers: { "Authorization": "Bearer YOUR_API_KEY" } }
);
const data = await response.json();
console.log(data);
```
</CodeGroup>

## 3. Read the response

```json
{
  "total_clicks": 42809
}
```

That's it. You can also pass `?period=24h` to get clicks from the last 24 hours only.
