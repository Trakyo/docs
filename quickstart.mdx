## 2. Make your first request

<CodeGroup>
```bash curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.trakyo.io/v1/links"
```

```python Python
import requests

response = requests.get(
    "https://api.trakyo.io/v1/links",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.trakyo.io/v1/links",
  { headers: { "Authorization": "Bearer YOUR_API_KEY" } }
);
const data = await response.json();
console.log(data);
```
</CodeGroup>

List endpoints use Stripe-style cursor pagination. Pass the last result's `{domainId}:{key}` as `starting_after` to fetch the next page, and check `has_more` to know when you're done.

## 4. Create a short link

<CodeGroup>
```bash curl
curl -X POST "https://api.trakyo.io/v1/links" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/my-page"}'
```

```python Python
response = requests.post(
    "https://api.trakyo.io/v1/links",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={"url": "https://example.com/my-page"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "https://api.trakyo.io/v1/links",
  {
    method: "POST",
    headers: {
      "Authorization": "Bearer YOUR_API_KEY",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({ url: "https://example.com/my-page" })
  }
);
const data = await response.json();
console.log(data);
```
</CodeGroup>
